SISTEMA DE GESTIÓN DE FABRICACIÓN

INTRODUCCIÓN

Este proyecto fue concevido para realizar el seguimiento de los distintos elementos a fabricar en una empresa. Cada elemento que se envíe a fabricar está acompañado de una orden de trabajo. A su vez, cada elemento podrá estar compuesto por una o más piezas.


ORDENES DE TRABAJO

Cada orden de trabajo cuenta con un cliente, un nombre de proyecto, y fecha de entrega. Además, al registrar un nuevo proyecto, se registra el usuario que realizó la carga o última modificación y la fecha.


PARTES

Cada orden de trabajo está compuesta por partes, que son las piezas constructivas necesarias para ejecutar la orden de trabajo. Cada pieza cuenta con un nombre, código de identificación y cantidad. El código de identificación se utiliza para vincular la pieza con los planos de frabricación.
